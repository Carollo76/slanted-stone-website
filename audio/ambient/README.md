# Ambient (fallback)

**Vibe:** generic, mood-agnostic. The picker falls back to this folder
when the requested mood folder (`fireplace/`, `forest/`, etc.) is empty.

Drop 1–2 versatile, neutral instrumental tracks here so we always have
*something* even when a specific mood hasn't been stocked yet.

**Good search terms:** "ambient", "background instrumental", "soft pad",
"neutral piano".

See `../README.md` for licensing rules.
